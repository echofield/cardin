# Cardin

Système de fidélité wallet pour commerces physiques.

## Produit

- Landing page de conversion en français (`/landing`)
- Calculateur de perte/récupération en temps réel
- Moteur 4 étapes (`/engine`) avec templates métier
- Cartes préconfigurées: Café, Restaurant, Boulangerie, Coiffeur, Institut / Beauté, Boutique

## Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS

## Développement

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```
